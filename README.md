### `// whoami`

I wrote my first algorithms in a notebook at 10 вҖ” no computer to run them on, just the ideas. Still build that way: idea first, machine second.

Backend is my craft, **Go** is what I reach for. gRPC, microservices, realtime services вҖ” the "how do all these pieces talk without falling over" problem is the part I actually enjoy. When a project needs the whole thing, I build the rest too: **Laravel**, **Flutter**, **Next.js**. One person, idea to running app.

The one rule I don't break: **it has to actually run.** Deployed, working, in real hands вҖ” not a demo rotting in a branch.
